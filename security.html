🛡️ Security

Last Updated: October 2025

We take security very seriously. Your ledger data is securely stored under your account. All sensitive operations use encrypted connections.

🔐 Authentication & Access Control

  • Phone OTP Verification: Phone number verification is required for login
  • Firebase Authentication: Industry-standard authentication system
  • Session Management: Automatic logout on suspicious activity
  • Data Isolation: Every user's data is completely isolated

🔒 Data Encryption

  • HTTPS/TLS: All network requests are encrypted
  • At-Rest Encryption: Data in Firebase is stored in encrypted form
  • In-Transit Security: End-to-end encryption during data transfer
  • Secure Backup: Cloud backups are also encrypted

🎤 Voice Entry Security

  • No Audio Storage: No audio recording is stored
  • Local Processing: Voice-to-text is processed on-device
  • Minimal Data: Only parsed text (name, amount) is saved
  • Optional Feature: You can disable this anytime

📱 App Permissions Security

  • Minimal Permissions: Only essential permissions are requested
  • Scoped Access: Photos permission is limited to selected images
  • No Background Access: The app does not access data in the background
  • Transparent Usage: Every permission's purpose is clearly explained

🔍 Regular Security Measures

  • Code Reviews: Regular security audits
  • Dependency Updates: Latest security patches applied
  • Vulnerability Scanning: Automated security checks
  • Incident Response: Quick response to security issues

✅ Your Security Controls

  • Logout: Log out from any device
  • Delete Data: Delete your data anytime
  • Export Data: Export your data to keep a backup
  • Account Deletion: Request full account deletion
